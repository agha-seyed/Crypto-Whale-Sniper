## 🇬🇧 English Guide
This powerful software is designed and developed by the **Wirangar Group (Programming, AI & Robotics)**. It serves as an ultra-professional tool for cryptocurrency traders to track massive on-chain movements and execute high-speed copy-trades.

**Core Features:**
- **Multi-Chain Sniper:** Execute ultra-fast token swaps on Ethereum, BSC, and Arbitrum through major DEXs (Uniswap, PancakeSwap).
- **Mempool Sniper:** Tracks unconfirmed transactions in the mempool before they are mined, allowing for front-running and copy-trading whales instantly.
- **Military-Grade Security:** User private keys are symmetrically encrypted in the database using the advanced `Fernet` algorithm.
- **VIP Payment System:** Integrated with the official Telegram Stars gateway for premium subscription handling.
- **Microservice Architecture:** Fully dockerized with `PostgreSQL` and `Redis` for high availability and performance.
